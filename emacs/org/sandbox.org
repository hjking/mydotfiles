| TAB       | re-align the table, move to next field                 |
| S-TAB     | re-align the table, move to previous field             |
| RET       | re-align the table and move to next row.               |
|           |                                                        |
| M-LEFT    | move current column left                               |
| M-RIGHT   | move current column right                              |
| M-S-LEFT  | kill the current column                                |
| M-S-RIGHT | insert a new column to the left of the cursor position |
| M-UP      | move current row up                                    |
| M-DOWN    | move current row down                                  |
| M-S-UP    | kill current row                                       |
| M-S-DOWN  | insert a new row above current row                     |
| C-c  -    | insert a horizontal line below current row             |
| C-c RET   | insert a horizontal line below current row with cursor |
| C-c ^     | sort the table lines in the region                     |
